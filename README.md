# ESP32 Door Monitor IoT 🚪

自宅の玄関ドアの開閉を検知し、リアルタイムでメール通知とGoogleスプレッドシートへのログ蓄積を行うIoTシステムです。

## 概要 (Overview)
ESP32とリードスイッチを用いてドアの状態変化（エッジ）を検知し、Google Apps Script (GAS) で構築したWeb APIを経由して通知と記録を行います。ハードウェアの配線からクラウド連携までを一気通貫で構築しました。

## システム構成 (Architecture)
- **エッジデバイス:** ESP32 DevKitC
- **センサー:** 磁気リードスイッチ (SPS-320)
- **バックエンド/通知:** Google Apps Script, Gmail, Google Sheets API
- **言語:** MicroPython (ESP32), JavaScript (GAS)
- **電源:** USB ACアダプター（常時設置・稼働）

## 工夫した点・課題解決 (Key Learnings)
開発において、以下の課題をソフトウェア・ハードウェア両面からのアプローチで解決しました。

1. **エッジ検知によるAPI負荷軽減**
   単純なポーリングではドアが開いている間APIが連続で呼び出されてしまうため、前回と現在の状態を比較し、「閉」から「開」へ遷移した瞬間のみを検知するエッジ検知アルゴリズムを実装しました。
2. **ハードウェアの電力仕様と要件定義**
   初期のプロトタイプではモバイルバッテリーで稼働させましたが、マイコンの消費電力が少なすぎるためにバッテリーのオートパワーオフが作動し、システムがダウンする問題に直面しました。最終的な要件である「常時稼働」に立ち返り、速やかにACアダプター給電へと設計変更することで安定稼働を実現しました。

## 使い方 (Usage)
1. `main.py` 内の `WIFI_SSID`, `WIFI_PASSWORD`, `GAS_URL` をご自身の環境に合わせて変更し、ESP32に書き込みます。
2. `gas_webhook.js` をGoogle Apps Scriptに貼り付け、Webアプリとしてデプロイします。
